## Course Syllabus

### Fundamentals & Complexity Analysis
- Algorithm complexity (time and space)
- Big O, Ω, Θ notation
- Divide and conquer strategies
- Euclidean algorithm, QuickSort
- Sieve of Eratosthenes

### Sorting Algorithms
- Quadratic algorithms: Insertion, Selection, Bubble Sort
- O(n log n) algorithms: Merge Sort, Quick Sort
- Linear sorting: Counting Sort, Radix Sort

### Data Structures
- **Basic Structures**: Stack, Queue, Deque
- **Linked Lists**: Single, double, circular, skip lists
- **Trees**: Binary trees, Prüfer code, tree diameter
- **Heaps**: Heap sort, priority queues
- **Binary Search Trees**: BST operations, order statistics
- **Balanced Trees**: AVL trees, random balancing
- **Advanced**: Fenwick Tree, Segment Trees, 2D Segment Trees

### Hashing & Sets
- Hash functions
- Collision resolution
- Open addressing
- Disjoint Set Union (DSU)

### Range Queries
- Prefix sums
- Fenwick Tree
- Sqrt decomposition
- Range Minimum Query (RMQ)
- Segment trees with lazy propagation

### Greedy Algorithms
- General greedy approach
- Scheduling problems
- Activity selection
- Huffman coding

### Dynamic Programming
- Overlapping subproblems
- Longest Common Subsequence (LCS)
- Longest Increasing Subsequence (LIS)
- Knapsack problem
- Pseudo-polynomial time

### Graph Algorithms
- Graph representation
- **Traversal**: BFS, DFS
- **Topological sorting**
- **Strongly connected components**
- **Euler paths and cycles**
- **Hamilton cycles**
- **Bridges in graphs**
- **Minimum Spanning Trees**: Kruskal's, Prim's algorithms
- **Shortest Paths**: 
  - Single source: Dijkstra's, Bellman-Ford
  - All pairs: Floyd-Warshall

### String Algorithms
- Naive string search
- Rabin-Karp algorithm
- Z-function
- Knuth-Morris-Pratt (KMP)
- Trie data structure
- Suffix arrays

### Computational Geometry
- Sweep line method
- Closest pair of points
- Convex hull: Graham scan, Jarvis march

### Advanced Topics
- Lowest Common Ancestor (LCA)
- Binary lifting
- ±1 RMQ algorithm
- NP-completeness
- NP-hard problems

## Repository Structure

Each assignment folder contains:
- `README.md` - Problem descriptions or links
- Solution implementations (`.cpp` files)

## Building and Running

Solutions are written in C++. To compile:
```sh
g++ -std=c++17 -O2 solution.cpp -o solution
./solution
```
